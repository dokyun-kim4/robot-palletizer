# Robot Palletizer
Fundamentals of Robotics final project   
by Dexter Friis-Hecht & Dokyun Kim

## Project Introduction
The goal of this project was to further explore the fields of both manipulation
and perception in the context of warehouse operations. We particularly focused on the palletization
problem, since we are both interested in the planning and optimization required for a robot to succeed
in this space. In addition, warehouse logistics is a huge space in robotics right now, and this project
provides a nice introduction for us to get familiar with the field.

## Setup and run instructions
This repository uses the 7DOF Gen3 robot arm platform with an overhead Intel RealSense camera. For connecting to the Kinova arm, reference [SETUP.md](https://github.com/dokyun-kim4/robot-palletizer/blob/main/SETUP.md#kinova-arm-documentation). Once you have 
```
Testing Environment...

Environment is ready to go

Have fun using the Kinova Robot Arm!
```
showing in your terminal, you are ready to run the palletizing routine. Ensure the RealSense camera is plugged into your computer and run the following command.
```
python3 palletizer.py
```
